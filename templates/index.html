Bienvenido a IDGS805 - Intro Flask

Este proyecto es una introducción al desarrollo web utilizando el framework Flask de Python.

En esta aplicación, encontrarás varias funcionalidades que te ayudarán a entender cómo construir aplicaciones web con Flask.

  • Calculadora: Una simple calculadora para realizar operaciones básicas.
  • Cinépolis: Una simulación de compra de boletos de cine.
  • Productos: Un listado de productos con opciones para editar y eliminar.
  • Alumnos: Un formulario con Flask-WTF y macros para agregar alumnos.
